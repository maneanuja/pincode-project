# Pincode Project

## Overview
This project uses Google Sheets, AppSheet, and Google Apps Script to fetch Indian pincode details using the India Post API.

## Technologies Used
- Google Sheets
- AppSheet
- Google Apps Script
- India Post API

## Features
- User enters a pincode.
- Calls the India Post API.
- Fetches District, State, and Region.
- Stores the results in Google Sheets.
